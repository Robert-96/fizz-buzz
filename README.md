# fizz-buzz

> **Note**: This is a demo repository.

Fizz-Buzz is a group word game for children to teach them about division.

## Tests

### Unit

```
$ python -m pytest tests
```

### Doctests

```
$ python -m pytest app --doctest-modules
```

### Tox

```
$ python -m tox
```
